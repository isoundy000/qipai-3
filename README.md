棋牌赛事大厅
===========================


## 简介

* 本项目是棋牌赛事大厅。
* 服务器使用typescript框架mydog（<https://github.com/ahuangege/mydog>）开发，客户端使用unity开发。
* 服务器运行环境：nodejs,mysql,redis。
* 客户端运行环境：unity。
* 由于个人精力和一些其他原因，暂不再继续开发。
* 对项目结构和代码有疑问的，可加QQ群：875459630。
